## 🚀 Como executar os exercícios

```bash
# Clone o repositório
git clone https://github.com/GustavoKozan/University_Exercises.git

# Entre na pasta desejada
cd numpy

# Execute o arquivo Python (se for .py)
python nome_do_arquivo.py
```

> Os arquivos `.ipynb` podem ser abertos diretamente no
> [Google Colab](https://colab.research.google.com/) ou no Jupyter Notebook.
